JPG Vs GIF

A JPG file saves photos and contains photo files, this format is the most popular way to save images. A GIF file saves photos and usually contains photo files, this format is commonly used for images or short animations.

Format History

The JPEG was created in 1992 by The Joint Photographic Experts Group. Similar files around this time were GIF and PNG. The JPG is still being used today and is considered an open source. The GIF file on the other hand was created around 1987 along with the JPG and the PNG by the CompuServe company. The GIF is also still used today and is an open source as well(Fisher).

Compresssion

The GIF file is a lossless compression format with a file size of 500Kb before compression and 50Kb after. On average, it compresses a file by 90%(Goodnight). The JPG file is a lossy compression format with a file size of 500Kb before compression and 75Kb after. On average, it compresses a file by 85%(Fisher).

Similar Formats and Advantages / Disadvantages

Some similar formats are TIFF, PNG, and RAW. Since GIF files are usually limited in the number of colors they can display, other graphic formats like JPG, which can store many more colors, are usually used for full images like those created with a digital camera. GIF files, then, are used on websites when there doesn’t need to be a massive range of colors, like for buttons or banners(Fisher). Some advantages they have over similar formats are that JPG and GIF are that GIF can represent transparency into an image, and JPG can compress the most amount of color without losing quality on the image.
